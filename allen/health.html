Health Page Instructions

Today we are going to build a basic web page on a health topic. This page and the info we find can be used for our Health Website Project.

1. Create an html file and call it health.html

- If you need help remembering how to start an html file search on google "html basic example"

2. Make sure to us following html elements in your health.html file:

<html></html>
<title></title> (call it 'Health')
<body></body>
<div></div>
<h1></h1> (Pick a disease that you are interested in or know someone who has been affected by it)
<p></p> (Definition of that disease)
<strong></strong> (add strong tags around the most important part of the disease definition)
<img /> (Add an image of what the disease may look like in a person or community)
<a></a> (link to where you got the definition. Add the site's title between the opening <a> and closing </a>)

3. Add and <hr> element to your page below the <a> tag.

- A hr tag adds a horizontal line on your page

4. Add the following 3 websites to your health.html file

- http://healthindicators.dshs.texas.gov/
- http://www.dshs.state.tx.us/
- http://mayoclinichealthsystem.org/
