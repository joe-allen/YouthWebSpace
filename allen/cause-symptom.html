Causes and Symptoms Page Instructions

Today we are going to build 2 basic web pages on the causes and symptoms of our chosen health topic.

1. Create an html file and call it causes.html

2. Create an html file and call it symptoms.html

3. Add the basic html elements to both pages to give you a guideline

- If you need help remembering how to start an html file search on google "html basic example"

4. Use the information you found last class to fill in both Causes and Symptoms html pages

- Don't forget to add header elements for both sections, causes and symptoms (<h1>)

- Use "ordered lists" and "un-ordered list"

- Use links (<a href="">) when citing your sources or referring to another site online for information

- Use your image elements to add images, or the strong element to make important text bold

5. Email your files when you are finished
