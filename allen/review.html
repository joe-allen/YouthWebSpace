March 23st, 2016 - Class 2

Basic html elements (a.k.a. 'tags')

1. Some of the most common used tags are:

paragraph: 
<p></p>

h1: 
<h1></h1>

h2: 
<h2></h2>

h3: 
<h3></h3>

a href: 
<a href=""></a>

img src: 
<img src="" />

View Example »

2. To make an Ordered List in html we use the <ol> tag

All <ol> tags will have <li> tags in them. For example:


<ol>

  <li>List Item 1</li>
  <li>List Item 2</li>
  <li>List Item 3</li>

</ol>

View Example »

3. Links and images are very important in the web. To create a link use the <a> tag with the 'href' attribute

To create an image use the <img> tag with the 'src' attribute

For example:

<a href="https://www.whitehouse.gov">Link To White House</a>

<img src="http://instagram.com/instagram.jpg" /> 

View Example »

March 21st, 2016 - Class 1

How to make a basic html page

1. All web pages have to start with element:
<!DOCTYPE html>

View Example »

2. Almost ALL HTML elements that are opened need to be closed. (Between the opening and closing angled brackets (> and <), is where visible html markup will be.

For example:

opened: 
<title>
closed: 
</title> (notice the closing slash)
example: 
<title>My Title</title>

View Example »

3. All web pages will have this basic code structure:


  <!DOCTYPE html>

  <html>

  <head>

  <title></title>

  </head>

  <body>

  ... code ...

  </body>

  </html>
