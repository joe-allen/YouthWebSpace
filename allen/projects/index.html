Our Health Projects!

Like we mentioned on our homepage, we will be using a very popular website framework to assist us in rapidly developing a nice, user friendly website.

The framework we're going to use is called "Bootstrap" and is developed by the designers and developers at Twitter.

To start, we'll choose from one of Bootstrap's sample pages:

Easy Sample » Hard Sample »

Instructions

1. Download your code using the download link next to your name

2. Unzip the download and find the "MyHealthProject" folder

3. In the "MyHealthProject" folder you will see the sample code for the "Easy" and "Hard" examples. Choose one and open it

4. Copy the code to a new file and save it as "index.html". (Make sure you save it in the "MyHealtProject" folder). This will be your projects homepage

5. Now, open your health.html file your created and update your index.html file with the code you have already written in your health.html file

6. Make sure to save and view the index.html file in your browser as you update it
