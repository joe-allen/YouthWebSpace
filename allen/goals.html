Project Goals:

Instructor will monitor and ensure the website meets specifications as outline in the scope of work. Candidate will work with AFN management team to create and implement curriculum to teach students the following:

  • To create a health care information website and introduce students to web development as a possible career path
  • Basics of HTML and PHP languages
  • how to create and manipulate Cascading Style Sheets (CSS)
  • Understand/ describe and use Bootstrap
  • Write scripts and programming code as needed to support the webpage
  • How to create online forms
  • How to successfully migrate a website between locations
  • How to research a topic and design a webpage that can effectively deliver that topic
